# Funded Next — $10k Stellar Instant

Funded Next does not offer a standard $10,000 evaluation challenge. The Stellar 2-Step, 1-Step, and Lite challenge tiers jump from $6,000 directly to $15,000. The $10,000 account size is available exclusively through the **Stellar Instant** product, which provides immediate funded account access without a challenge evaluation phase.

The Stellar Instant $10,000 account costs $299.99. Traders log in and begin trading immediately upon account setup, with no profit targets to pass beforehand.

## Account Parameters

| Parameter | Value |
| --- | --- |
| Price (USD) | $299.99 |
| Account Type | Instant funded (no evaluation phase) |
| Profit Withdrawal Trigger | 5% account growth |
| Max Loss (Trailing) | 6% of peak balance |
| Max Daily Loss | None |
| Minimum Trading Days | None |
| Time Limit | None |
| Initial Payout Split | 70% |
| Max Payout Split | 80% |
| Scaling | Up to $2,000,000 (account doubles every 10% growth + withdrawal) |
| Weekend Holding | Allowed |
| News Trading | Allowed (40% profit-count restriction applies in funded accounts) |

## Additional Details

- The 6% trailing maximum loss is calculated from the account's peak equity, not the initial balance.
- There is no daily loss cap on the Stellar Instant account; only the overall trailing drawdown applies.
- Account scaling doubles the allocation each time the trader achieves 10% growth and makes a withdrawal, continuing up to $2,000,000 total allocation.
- Reset fees are discounted 10% from the original plan cost if a trader needs to restart.
- No KYC (Know Your Customer) verification is required before trading begins; KYC is completed before the first withdrawal.
