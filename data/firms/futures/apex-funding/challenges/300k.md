# Apex Trader Funding — $300k Account (Legacy)

The Apex Trader Funding $300,000 evaluation account was a product available under the legacy pricing model (pre-March 2026). Following the 4.0 update in March 2026, this account size is no longer available for new purchase. Traders who purchased a $300K evaluation before the 4.0 transition may continue to hold it under the legacy terms.

## Legacy Account Parameters

The following parameters reflect the legacy model. Verify with Apex Trader Funding support for current legacy account status.

| Parameter | Value |
| --- | --- |
| Historical Monthly Fee | ~$677/month (legacy subscription model) |
| Profit Target | ~$20,000 |
| Trailing Drawdown | ~$7,500 |
| Trailing Drawdown Type | EOD or Intraday (depended on selected variant) |
| Profit Target (% of account) | ~6.7% |
| Minimum Trading Days | 5 |
| Payout Split | 100% |
